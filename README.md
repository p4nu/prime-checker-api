### 2. `npm install`

### 3. Run the project!

#### For PowerShell (Recommended, but no matter):

`PS> $env:DEBUG='myapp:*'; npm start`

#### For Mac or Linux users:

`$ DEBUG=myapp:* npm start`

#### For Windows command prompt:

`> set DEBUG=myapp:* & npm start`
